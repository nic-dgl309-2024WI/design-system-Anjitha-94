Navbar for Outback Nursery

Demo Site
Product list FAQ & Gift certificates About Contact

Classes used in nav bar

topnav :css for top navbar
u-color-secondary-bg: Back ground Color
u-font-lora :Font used for navbar
Also I used javascript code to make the nav bar responsive

Html code 



<section>
                <h1 class="u-font-merriweather u-font-size-600 u-color-primary-text003 l-flex l-flex-row l-justify-center l-p-4-50">
                    Navbar for Outback Nursery
                </h1>
                <div class="l-flex l-justify-center l-align-items-center l-flex-column l-p-1 ">
                    <!-- Top Navigation Menu -->
                    <!--The code taken from w3 schools:https://www.w3schools.com/howto/howto_js_mobile_navbar.asp make the class names according to BEM --> 
                    <div class="topnav u-color-white u-color-secondary-bg u-font-lora">
                        <div class="topnav__logo-container u-color-secondary-bg">
                            <a href="index.html" class="topnav__logo topnav__logo--active topnav__link active">Demosite</a>
                            <!-- "Hamburger menu" / "Bar icon" to toggle the navigation links -->
                            <a href="javascript:void(0);" class="topnav__icon" onclick="myFunction()">
                                <i class="fa fa-bars"></i>
                            </a>
                            <!-- Navigation links (hidden by default) mobile screen-->
                            <div class="topnav__links" id="myLinks">
                                <a href="#" class="topnav__link">Product list</a>
                                <a href="#" class="topnav__link">FAQ & Gift certificates</a>
                                <a href="#" class="topnav__link">About</a>
                                <a href="#" class="topnav__link">Contact</a>
                            </div>
                        </div>
                    </div>
                </div>
            </section>
